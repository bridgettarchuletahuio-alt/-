# 链接合并管理台

这个版本按后台管理系统方向实现，风格接近短网址平台控制台。

## 功能

- 左侧导航 + 顶部栏 + 指标卡
- 新增链接合集短网址
- 每行一个链接，自动校验并去重
- 列表支持搜索和排序
- 支持修改、删除、复制、打开短网址
- 访问短网址后进入合集页，并累计点击数
- 本地数据持久化到 localStorage

## 本地运行

1. 进入项目目录并启动静态服务

python3 -m http.server 8000

2. 打开浏览器

http://localhost:8000

## 短网址格式

示例：

http://localhost:8000/#go=abc123

说明：

- go 后是自动生成的短码
- 打开后会进入该合集页
- 每次打开会为该记录点击数 +1

## 注意

- 这是纯前端演示，不含后端数据库
- 不同浏览器或清理缓存后，localStorage 数据可能丢失
- 一键连续打开可能被浏览器拦截部分弹窗
